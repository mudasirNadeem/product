# Login System with Bun

## Setup

1. Install Bun (Windows):
```powershell
powershell -c "irm bun.sh/install.ps1 | iex"
```
2. Upgrade Bun (Windows):
```powershell
bun upgrade --canary
```
3. Start server:
```bash
bun run server.js
```

4. Access application:
- Open `http://localhost:3000`
- Login credentials:
  - Username: `demo`
  - Password: `demo123`
